WebGL draft extensions

For a smoother appearance of cylinders and spheres your browser needs to have the *EXT_frag_depth* extension available. The WebGL Report should list the extension if active. If not, you can enable WebGL draft extensions following these instructions:
  • Chrome: browse to about:flags, enable the *Enable WebGL Draft Extensions* option, then relaunch.
  • Firefox: browse to about:config and set *webgl.enable-draft-extensions* to *true*.

  • Internet Explorer: Currently, the *EXT_frag_depth* extension is not supported.
