libxdrfile2

For the efficient access of GROMACS trajectory files, the library *libxdrfile2* needs to be installed. cd lib/ sudo python setup.py install make sure you have the python development files installed sudo apt-get install python-dev

Configuration files

The website can be configured through some javascript files that should be set-up.

Copy the sample tracking.js file and add any tracking code you need. cp js/tracking.js.sample js/tracking.js Copy the sample examples.js file and add any examples you want to be available on your server. cp js/examples.js.sample js/examples.js
