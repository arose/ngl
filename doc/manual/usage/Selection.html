[name]

Selections (or 'Sele' for short) can be input at various places in the [page:Interface GUI] or when scripting. They are used to limit which atoms/residues are shown in a [page:Representation Representation] or what atoms are loaded from a [page:Trajectory].

Language

Expressions

  • keywords: *backbone*, *protein*, *nucleic*, *hetero*, *all*
  • chain name: :A
  • atom name: .CA, .C, .N, ...
  • model: /0, /1, ...
  • residue name: *ALA*, *GLU*, *SOL*, ...
  • residue number: *1*, *2*, *100*
  • residue number range: *3-40*

Logical operators (in order of binding strength)

  • *NOT*
  • *AND*
  • *OR*
Additionally, parentheses *()* can be used for grouping.
