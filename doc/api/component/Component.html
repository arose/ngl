[name]

A container that contain a data object and can be added to a [page:Stage]. For [page:Structure] objects there is [page:StructureComponent] and for [page:Surface] objects [page:SurfaceComponent].

Example

var structure = new NGL.Structure(); var sele = "ALL"; var structureComponent = new NGL.StructureComponent( stage, structure, sele );
